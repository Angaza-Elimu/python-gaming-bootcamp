# Python Programming Bootcamp - 5 Day Curriculum

A comprehensive Python programming bootcamp designed for high school beginners. This repository contains complete lesson plans, code examples, and three fully functional Pygame projects.

This 5-day intensive bootcamp takes students from zero programming experience to building their own games. Each day builds on the previous, with hands-on projects and real-world examples.

### Day 1: Getting Started & Python Fundamentals
- Introduction to programming concepts
- Installing Python & VS Code
- Variables and data types (strings, integers, floats, booleans)
- User input and output
- Arithmetic operations
- **Projects**: Greeting Bot, Math Quiz, Simple Calculator

### Day 2: Decisions, Logic & Loops
- Conditional statements (if/elif/else)
- Boolean logic and comparison operators
- For loops and while loops
- Loop control (break, continue)
- **Projects**: Login System, Grade Calculator, Rock Paper Scissors

### Day 3: Working With Collections
- Lists and list manipulation
- Tuples for immutable data
- Dictionaries and key-value pairs
- Nested data structures
- **Projects**: Contact Book, RPG Inventory System

### Day 4: Functions & Code Organization
- Defining and calling functions
- Parameters and return values
- Code organization best practices
- Basic debugging techniques
- **Projects**: Password Generator, Interactive Quiz App, To-Do List Manager

### Day 5: Introduction to Game Development
- Installing Pygame library
- Understanding the game loop
- Drawing shapes and handling input
- Collision detection
- **Projects**: Catch the Falling Objects game
